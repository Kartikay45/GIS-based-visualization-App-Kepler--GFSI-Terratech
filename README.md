(https://github.com/Kartikay45/Dataset-Terratech-GIS-Hackathon.git)

### Local dev

```
npm install
```

or

```
yarn --ignore-engines
```

add mapbox access token to node env

```
export MapboxAccessToken=<your_mapbox_token>
```

then

```
npm start
```
